## 编译

1. 将本项目完整克隆
2. cd 到项目目录
3. 用XeLaTeX进行编译，请保证assists文件夹位于同级且没有损坏
4. 忽略错误：<code>LaTeX: `!h' float specifier changed to `!ht'.</code>

**注意：release中的pdf是在5.15.167.4-microsoft-standard-WSL2中编译、已配置使用Windows字体。**
